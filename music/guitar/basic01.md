### Popular notations for plucking fingers

```ASCII
                    M
                I  /"\  A
               /"\|\./|/"\        p = pulgar (thumb)
              |\./|   |\./|
              |   |   |   | X     i = indice (index)
              |   |>~<|   |/"\
              |>~<|   |>~<|\./|   m = medio (middle)
           P  |   |   |   |   |
          /~T\|   |   =[@]=   |   a = anular (ring)
          |_/ |   |   |   |   |
          |   | ~   ~   ~ |   |   x = pinky
          |~< |             ~ |
          |   '               |
          \                   |

    /\---Pick
```
